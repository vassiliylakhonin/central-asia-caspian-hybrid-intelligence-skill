# Agent-Eval Delta Is the Hard Bar 2 Gate

This repo is built primarily for agents and agent integrators, so Bar 2 is defined as agent-validation rather than practitioner validation. The hard Bar 2 evidence is a set of with/without agent-eval deltas showing how the Agenda Intelligence product shell changes output structure when this specialist is routed in: provenance, uncertainty, evidence mode, watch-next indicators, role-based actions, and regional mechanism depth. Practitioner review remains valuable, but it is an optional trust layer for practitioner-facing adoption, not the hard integration gate.

**Consequences**

- Bar 2 can be cleared with self-scored structural agent-evals if they are honest, reproducible, and explicitly non-factual.
- Agent-evals must not be presented as factual verification, model-quality comparisons, aggregate benchmarks, or practitioner approval.
- Practitioner reviews should live separately under `validated-cases/` when the audience needs buying-side domain trust.
